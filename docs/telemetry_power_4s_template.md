# WebKurierDrone • Telemetry + Power + Solar Template (4S Platforms)

Единый «бортовой стандарт» для дронов WebKurier:
1. systemd-юнит для авто-старта телеметрии на SBC (Jetson / Pi / x86).
2. Логика разводки питания для секций + BMS + 220 V зарядка.
3. Варианты солнечных контроллеров под 4S (гибкие панели).
4. Шаблон перерасчёта под конкретные датчики и кодеки.
